# 在线协作表格

实时多人协作在线电子表格，支持多人同时编辑、单元格合并、中文输入等功能。

## 功能特性

### 基础功能
- 🔄 实时协作 - 多人同时编辑，数据实时同步
- 👥 光标显示 - 实时显示其他用户的光标位置和用户名
- 💾 自动保存 - 数据自动持久化到 SQLite 数据库
- ⌨️ 快捷键支持 - 支持方向键导航、Tab/Enter 切换单元格
- 🎨 用户标识 - 每个用户有独特的颜色标识
- 🌏 中文输入 - 完美支持中文输入法

### 表格功能
- 📝 单元格编辑 - 点击选中后直接输入，双击或按 Enter 进入编辑模式
- 🔗 单元格合并 - 支持选择多个单元格进行合并
- ✂️ 取消合并 - 支持取消已合并的单元格
- 🖱️ 框选支持 - 支持鼠标拖拽选择多个单元格
- ⬆️⬇️⬅️➡️ 方向键导航 - 使用方向键在单元格间移动
- 🗑️ 批量删除 - 支持选中多个单元格后批量清除内容
- 📋 复制粘贴 - 支持 Ctrl+C/X/V 快捷键和右键菜单
- 🖱️ 右键菜单 - 支持剪切、复制、粘贴、清除等操作

### 格式化功能
- 🎨 文字颜色 - 修改单元格文字颜色
- 🖌️ 背景颜色 - 修改单元格背景颜色
- 📏 字号设置 - 支持 10-32px 多种字号
- **B** 粗体 - 设置文字粗体 (Ctrl+B)
- **U** 下划线 - 设置文字下划线 (Ctrl+U)
- ⬅️↔️➡️ 对齐方式 - 左对齐、居中、右对齐

## 技术栈

- **前端**: Vue 3 + TypeScript + Vite
- **后端**: Node.js + Express + WebSocket
- **协作**: Yjs (CRDT 无冲突复制数据类型)
- **存储**: SQLite (better-sqlite3)

## 快速开始

### 安装依赖

```bash
npm run install:all
```

### 启动开发服务器

```bash
npm run dev
```

### 单独启动

```bash
# 启动后端 (端口 3001)
cd server && npm run dev

# 启动前端 (端口 5173)
cd client && npm run dev
```

## 使用说明

### 基本操作
1. 打开浏览器访问 `http://localhost:5173`
2. 点击单元格进行选择
3. 选中后直接输入文字即可开始编辑
4. 双击或按 Enter 键进入编辑模式
5. 按 Escape 键取消编辑
6. 按 Tab 键切换到下一个单元格
7. 使用 Shift + 点击或拖拽选择多个单元格

### 复制粘贴
1. 选中单元格或区域
2. 按 Ctrl+C 复制，Ctrl+X 剪切
3. 选中目标位置，按 Ctrl+V 粘贴
4. 也可以使用右键菜单进行操作

### 右键菜单
1. 在单元格上右键点击
2. 选择剪切、复制、粘贴或清除内容
3. 支持对选区进行批量操作

### 协作功能
1. 打开多个浏览器窗口访问同一地址
2. 在一个窗口中编辑，另一个窗口会实时同步显示
3. 可以看到其他用户的光标位置和用户名

### 单元格合并
1. 鼠标拖拽选择多个单元格（或按住 Shift + 点击）
2. 点击工具栏中的"合并单元格"按钮
3. 要取消合并，点击已合并的单元格，再点"取消合并"

### 格式化操作
1. 选中单元格或区域
2. 使用工具栏按钮进行格式化：
   - 点击 **B** 按钮切换粗体
   - 点击 **U** 按钮切换下划线
   - 点击颜色按钮选择文字颜色或背景颜色
   - 使用下拉框选择字号（10-32px）
   - 点击对齐按钮设置左对齐、居中或右对齐
3. 格式会应用到选中的所有单元格

### 快捷键
| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 方向键导航 |
| Shift + ↑ ↓ ← → | 扩展选区 |
| Enter | 进入编辑模式 / 确认编辑并向下移动 |
| Tab | 向右移动 |
| Shift + Tab | 向左移动 |
| Delete / Backspace | 清除选中内容 |
| Escape | 取消编辑 |
| Ctrl + C | 复制 |
| Ctrl + X | 剪切 |
| Ctrl + V | 粘贴 |
| Ctrl + A | 全选 |
| Ctrl + B | 粗体 |
| Ctrl + U | 下划线 |
